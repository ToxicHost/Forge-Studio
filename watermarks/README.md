# Watermarks

Drop watermark images here, then pick one in **Settings → Watermark**.

- Supported: `.png` and `.webp` (transparency strongly recommended — an
  opaque image will show its full rectangle).
- The selected watermark is composited onto every generated image as the
  final step, at the position / opacity / size / margin / rotation you choose.
- Files are matched by filename, so keep names simple (no subfolders).
